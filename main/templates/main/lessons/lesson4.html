Айнымалылар өте маңызды, себебі олар ақпаратты сақтауға және кейінірек пайдалануға мүмкіндік береді. Бастапқыда айнымалыға неліктен бірдеңе жазу керектігі анық болмауы мүмкін, егер сіз оларсыз жай ғана мәндермен жұмыс істей алсаңыз. Айнымалылар туралы түсінік сәл кейінірек, күрделірек бағдарламалар жасауды бастағанда және ақпаратты бір жерде сақтау қажет болғанда келеді.


Python тіліндегі айнымалы типтер нақты жарияланбаған, бірақ олар сонда да бар. Аудармашы айнымалыға не жазылғанын түсінеді және осының негізінде осы айнымалыға тип қосады.


Программаны орындау барысында айнымалыларды қайта жазуға, сонымен қатар олардың типін өзгертуге болады. Егер бастапқыда айнымалы float типінде болса, оны кейін басқа түрге түрлендіруге болады, мысалы, string.

first_num = 23.2 # Тип данных float
    first_num = "1" # Тип данных string
Бірнеше айнымалыларды әртүрлі деректер түрлерімен біріктіру кезінде бағдарлама қатені тудырады.


Мысалы:

first_num = "IloveYou"
    second_num = 13
    res = first_num + second_num # Скрипт выдаст ошибку
Python-да 4 негізгі айнымалы түрі бар:
  • some = 1 Integer - бүтін сандар;
  • some = 1,12 Float - өзгермелі нүкте сандары;
  • some = "Сәлеметсіз бе" String - жолдар;
  • some = Ақиқат Boolean - "жалған" немесе "шындық" мәнін қабылдайтын деректер түрі.
Басқа да түрлері бар, бірақ біз оларды келесі сабақтарда қарастырамыз.

Бір жолда бірден бірнеше айнымалылар жасауға болады:
first = sec = third = 1 # Всем трём переменным будет присвоено значение 1
    first, sec, third = "Hi", 75, 23.1 # Поочередное присвоение значений
