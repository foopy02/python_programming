Пайдалы сілтемелер:

  • Барлық модульдері бар сайт.


Модульдер - басқа Python файлдарының ішіне импортталатын қосымша файлдар. Әрбір модульде модульді кез келген файлға қосқаннан кейін пайдалануға болатын айнымалылар мен функциялар жиынтығы бар. 


Сіз өзіңіздің модульдеріңізді жасай аласыз немесе дайын Python модульдерін пайдалана аласыз. Сіз барлық стандартты модульдердің толық тізімін таба аласыз осы сілтеме бойынша.


Барлық модульдерді есте сақтау мүмкін емес, ең бастысы, бұл қажет емес. Сізге тек қай модуль не үшін жауапты екенін түсіну керек және қажет болған жағдайда Интернетте толтырылған құжаттаманы қарауға болады. Мысалы, мұрағат жасағыңыз келе ме немесе бар мұрағатқа бірдеңе қосқыңыз келе ме? Содан кейін сізге архивтермен жұмыс істеуге мүмкіндік беретін ZipFile модулі қажет: оларды жасауға, файлдарды қосуға, файлдарды жоюға және басқа әрекеттерді орындауға.


Мұндай модульдердің өте көп саны бар және олардың әрқайсысы кейбір функциялар мен инновацияларға жауапты. 


Модульді қосу үшін import пәрменін пайдаланып, модульдің атын көрсету керек.


Мысалы:

import time # Импорт одного модуля
    import random, os # Импорт нескольких модулей

Сіз өзіңіздің модульдеріңізді де жасай аласыз. Ол үшін жаңа файл жасап, оның аты бойынша файлды модуль ретінде басқа файлдарға қосу керек.


From командасы

Тек кейбір әдістерді немесе айнымалы мәндерді импорттау үшін from пәрменін пайдалануға болады. Бұл пәрмен келесідей жазылады:

from math import ceil # Импортируем лишь метод ceil


Модульдерге арналған бүркеншік аттар

Модульдермен жұмыс істеу кезінде бүркеншік аттарды да пайдалануға болады. Бұл модуль атын қысқартуға мүмкіндік береді, егер ол тым үлкен болса және оны әр кезде жазғыңыз келмесе.
