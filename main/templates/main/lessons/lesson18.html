Конструкторлар объект жасалған кезде бірден оның кейбір сипаттамаларын орнатуға мүмкіндік береді. Мысалы, сізде нысанда міндетті түрде болуы керек бірнеше айнымалылар бар. Конструктор жасауға және нысан жасалған кезде берілетін бірнеше параметрлерді көрсетуге болады. 


Бір сыныпта конструкторлардың шектеусіз саны болуы мүмкін және аудармашының өзі қай конструкторға сілтеме жасап тұрғаныңызды түсінеді. Конструктор жасау үшін __init__ кілт сөзін пайдалану керек.


Конструкторы бар сыныптың мысалы:

class Cars:
	wheels = 0
	marka = ""
	
	def __init__(self, wheels, marka):
		self.wheels = wheels
		self.marka = marka
bmw = Cars(4, "X3") # Сразу добавили характеристики
print (bmw.wheels) # Результат - 4
