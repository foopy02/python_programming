Индекстер

Тізімдердегі нөмірлеу нөлден басталады, себебі тізім негізінен жай массив болып табылады, әдеттегі массивтегідей, санау 0-ден басталады. Индекстегі бірінші элемент 0, екіншісі - 1, үшінші - болады. 2 және т.б. Егер біз жоқ элементті алуға әрекет жасасақ, ол қатеге әкеледі.


Мысалы:

a = [0, 23, "Hi"] # Список
    print (a[4]) # Выдаст ошибку, так как элемента не существует

Python тілінің ыңғайлы ерекшелігі - теріс индекстерді пайдаланып элементтерді соңынан алу мүмкіндігі. Мысалы, егер бізге екінші элемент соңынан қажет болса, оны келесідей жаза аламыз:

a = [0, 23, "Hi", 1.56, 9] # Список
    print (a[-2]) # Будет выведено 1.56


Кесінділер

Шектегіштер сізге қажет элементтерді ғана алу арқылы тізімді қысқартуға мүмкіндік береді. Олар келесідей жұмыс істейді: тізім[START:END:STEP].

  • Бастау - қай элементтен бастау керек (әдепкі - 0);
  • Соңы - қандай элемент арқылы элементтерді аламыз (әдепкі тізімнің ұзындығына тең);
  • Қадам - ​​элементтерді қандай қадаммен орындаймыз, мысалы, әрбір 2 немесе 3 (әдепкі бойынша, әрбір 1).


Бөлімде бір, бірнеше немесе тіпті барлық параметрлерді алып тастауға болады.

list[::3] # Берем каждый третий элемент
    list[2::2] # Начиная со второго элемента берем каждый второй элемент
    list[4:6:] # Начиная с 4 элемента берем все элементы по 6 элемент
    list[::] # Берем все элементы
Теріс сандарды кесінділер үшін де пайдалануға болады.
