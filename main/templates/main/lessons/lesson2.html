Пайдалы сілтемелер:

  • Ресми веб-сайт Python;
  • Бағдарлама PyCharm.


Әзірлеу ортасын орнату әрқашан көп уақытты қажет етеді, өйткені барлығын жүктеп алып, конфигурациялау қажет. Бұл оқулықта біз сізге Python бағдарламасын Windows жүйесінде орнату процесімен таныстырамыз.


Python-ды Windows жүйесіне қалай орнатуға болады?

Бірінші қадам - ​​компьютерге Python жүктеп алу. Мұны істеу үшін өтіңіз ресми сайт және Python бағдарламасының соңғы нұсқасын жүктеп алыңыз. Жүктеп алудың екі түрлі нұсқасы бар: Python 2 және Python 3. Курс жаңа нұсқаны қамтиды, сондықтан Python 3 нұсқасын жүктеп алуға болады.



Жүктеп алғаннан кейін орнатушыны екі рет басыңыз және Python орнатушысы іске қосылады. Ашылған терезеде Add Python X.Y to PATH құсбелгісін қойып, Python-ды орнату үшін "Орнату" түймесін басыңыз.



Python орнатудан кейін бірден пайдалануға дайын болады. Сізге тек оның өнімділігін тексеру керек болады.


Жаңа қалта Бастау қойындысында пайда болуы керек, онда IDLE іске қосылуы мүмкін. Егер мұндай қалта болмаса, бағдарламаны қолмен тауып көріңіз. 


IDLE — Python тілімен жұмыс істеуге болатын ендірілген әзірлеу ортасы. Сондай-ақ өнімділікті консольде (терминалда) тексеруге болады. Ол үшін консольді ашып, «python3» деп теріңіз. Егер ештеңе болмаса, жай ғана "python" сөзін теріп, Enter пернесін басып көріңіз.


Даму ортасы

Сіз әзірлеу ортасы ретінде кез келген қолайлы бағдарламаны пайдалана аласыз. Курс барысында біз ең сенімді бағдарламалардың бірі – PyCharm қолданбасын қолданамыз. 


Егер басқа бағдарламаны қаласаңыз, оны пайдалана аласыз. Ең бастысы, сіздің бағдарламаңыздың Python тілін қолдауы бар екеніне көз жеткізіңіз. 


Файлға жолды көрсету

Windows жүйесінде файлға апаратын жолды көрсету үшін келесі  құрылымның мысалын пайдалану керек: "C:\Users\USER\Downloads\FILE.py".
