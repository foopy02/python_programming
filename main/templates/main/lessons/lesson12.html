Функцияларды қайталанатын кодты қоюға және қажет кезде оған сілтеме жасауға болатын шағын ішкі бағдарламалар деп атауға болады. Функциялар бағдарламаларды құруды әлдеқайда жеңілдетеді, өйткені бір типтегі кодты бірнеше рет көшірудің қажеті жоқ, бірақ біз жай ғана бір жалпы функцияны пайдалана аламыз.


Көптеген адамдар функциялар мен әдістерді шатастырады және олардың арасындағы айырмашылықты түсінбейді. Іс жүзінде ешқандай айырмашылықтар жоқ, өйткені әдістер де, функциялар да бірдей. Кластардан тыс жазылатын функциялар функциялар, ал сыныптардың ішінде жазылатын функциялар әдістер деп аталады.


Жағдай айнымалылармен бірдей. Сыныптарда айнымалылар өрістер деп аталады, ал сыныптардан тыс олар айнымалылар деп аталады.


Python-да функциялар def кілт сөзі арқылы жасалады. Әрбір функцияның кейбір параметрлері болуы немесе мүлде болмауы мүмкін. Қажет болса, функциялар кодты орындау кезінде бір нәрсені қайтара алады.

def some_test(): # Функция без параметров
	x = 23
	return x # Функция возвращает что-либо

def some_test_2(a, b, c): # Функция с 3 параметрами
	return a * b * c # Функция возвращает результат умножение чисел

def print_something(word, prefix): # Функция с 2 параметрами
	print (prefix, "-", word)
	pass # Функция ничего не возвращает

Функция ештеңе қайтармағанда, pass кілт сөзі жазылуы керек. 


Функциялар басқа функцияларды қайтара алады, осылайша оларды шақырады. Функцияны шақыру үшін оның атын жазып, егер бар болса, параметрлерді беру керек:

res = some_test_2 (2, < span class="token number">67, 12 )

Жоғарыдағы мысалда функцияның орындалу нәтижесі res айнымалысына орналастырылады. Одан әрі айнымалымен бағдарламадағы қалыпты мән сияқты жұмыс істеуге болады.


Анонимді функциялар

Егер функция кодтың үлкен көлемін орындауды қажет етпесе, онда сіз анонимді функцияны пайдалана аласыз. Бұл lambda функциясын қажет етеді. 


"lambda" функциясын жасау мысалы:

multiple = lambda a, b: a * b
multiple(7, 12)

Бұл функцияның аты жоқ, бірақ оны кейінірек кәдімгі функция ретінде шақыру қажет айнымалыға тағайындауға болады.
